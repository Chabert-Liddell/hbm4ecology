hbm4ecology

Data, Help Function and Analysis for the Book Introduction to Hierarchical Bayesian Modeling for Ecological Data

This is the companion package for the book Introduction to Hierarchical Bayesian Modeling for Ecological Data (Parent and Rivot, 2012) . The package contains the datasets used in the book as well as some helper functions for rstan and rjags. The analysis of the datasets are provided as articles.
